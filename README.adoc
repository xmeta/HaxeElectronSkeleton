= Haxe Electron Skeleton

== Get started
. `git clone https://github.com/xmeta/HaxeElectronSkeleton.git`
. `cd HaxeElectronSkeleton`
. haxelib install
+
----
$ haxelib setup
Path : ./lib

$ haxelib install electron
----

== Build
`haxe build.hxml`

== Run
`haxe run.hxml`

== Release Build
. `npm install -g asar`
. `asar pack target/debug ./target/release/app.asar`
